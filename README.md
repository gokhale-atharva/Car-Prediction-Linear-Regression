# 🚗 SmartCar Pricing: A Web-Based Machine Learning Predictor

An end-to-end production-grade machine learning application designed to estimate fair-market automobile valuations. Built on real historical records from the CarDekho marketplace, this system features a regularized linear model pipeline integrated into a highly responsive, dynamic web dashboard.

---

## 🛠️ Tech Stack & Architecture

- **Core Analytics & ML:** Python, Scikit-Learn, Pandas, NumPy, Joblib
- **Web Frontend Engine:** Streamlit
- **Machine Learning Model:** Regularized Ridge Regression with Cross-Validation (`GridSearchCV`)
